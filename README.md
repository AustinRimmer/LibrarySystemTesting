
## Setup Instructions
  1. install visual studio 2019
  2. in the file directory open the terrminal and verify the instilations are correct by running:
    npm install
    npm install cypress --save-dev
    npm install express
## Running the Application
  1. Download code and open in visual studio 19
  2. Start the server: npm start
  3. Open browser: http://localhost:3000
## Running Cypress Tests
  ### Headless Mode
    npm run cypress:run
  ### Normal Mode
    npx cypress open
